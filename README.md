# Quiz Web App

A **browser-based interactive quiz application** built using **HTML, CSS, and JavaScript**. The app randomly selects questions from a predefined pool and allows users to test their knowledge with instant feedback.

## Features

* Randomly selects **5 questions** from a question pool
* Multiple-choice answers for each question
* Instant feedback (correct answer in **green**, wrong answer in **red**)
* Displays **final score** at the end of the quiz
* Shows the **correct answers** for all questions
* Option to **restart the quiz**

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

```
Quiz-Web-App
│
├── index.html
├── styles.css
└── scripts.js
```

## How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open **index.html** in any web browser

The quiz will start automatically.
